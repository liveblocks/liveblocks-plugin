# Liveblocks Agent Instructions

This repository provides agent guidance for building with Liveblocks and Yjs.

Use the bundled skills when they are relevant:

- `skills/liveblocks-best-practices/SKILL.md` for Liveblocks rooms, auth,
  presence, Storage, comments, notifications, text editors, and React patterns.
- `skills/yjs-best-practices/SKILL.md` for Yjs data modeling, providers,
  updates, performance, and common debugging issues.

If MCP tools are available, use the Liveblocks MCP server from `.mcp.json` only
with a development project secret key. Do not use a production secret key.
